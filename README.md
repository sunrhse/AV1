# Como rodar o projeto

## Pré-requisitos
- Node.js

## Instalação
Clone o repositório e acesse a pasta do projeto:
```bash
git clone <url-do-repositorio>
cd <nome-do-projeto>
```
Instale as dependências:
```bash
npm install
```

## Execução
Para rodar o sistema:
```bash
npm start
```

## Primeira execução
Na primeira execução, o sistema solicitará o cadastro do administrador.
